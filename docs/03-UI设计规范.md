# UI 设计规范

> 版本：v1.0 | 日期：2026-06-21 | 状态：初稿

---

## 1. 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| **主色调** | `#1A3C6E` | 深蓝，用于侧边栏、导航、主要按钮 |
| **强调色** | `#FF7A00` | 亮橙，用于关键操作按钮、数据高亮、图标 |
| **背景色** | `#F5F7FA` | 浅灰蓝，用于内容区背景 |
| **卡片背景** | `#FFFFFF` | 白色，用于信息卡片 |
| **文字主色** | `#333333` | 深灰，正文内容 |
| **文字辅助** | `#6C757D` | 中灰，次要信息 |
| **成功色** | `#28A745` | 绿色，成功状态、正向数据 |
| **警告色** | `#FFC107` | 黄色，警告状态 |
| **危险色** | `#DC3545` | 红色，删除操作、负向数据 |
| **边框色** | `#DEE2E6` | 浅灰，表格和卡片边框 |

### 2.2 侧边栏

| 属性 | 规范 |

| 属性 | 规范 |

## 3. 组件规范

### 3.1 按钮

| 类型 | 背景 | 文字 | 用例 |
|------|------|------|------|
| 主按钮 | `#1A3C6E` | `#FFF` | 确认、提交、保存 |
| 强调按钮 | `#FF7A00` | `#FFF` | 导入、导出、关键操作 |

## 4. 图表规范

使用 Chart.js，以下为默认配置：

| 属性 | 规范 |
|------|------|
| 折线图颜色 | `#1A3C6E`（主）|
| 柱状图颜色 | `#1A3C6E` 和 `#FF7A00` 交替 |
| 网格线 | 浅灰 `#E9ECEF`，虚线 |
| 图例位置 | 图表上方，水平排列 |
| 字体 | 12px，`#6C757D` |
| Tooltip | 白色背景，阴影边框 |

## 6. 响应式断点

| 断点 | 宽度 | 适配 |
|------|------|------|
| ≥1200px | Desktop | 侧边栏展开 |
| 768-1199px | Tablet | 侧边栏折叠 |
| <768px | Mobile | 侧边栏隐藏，汉堡菜单 |

## 7. 页面示例

### 7.1 登录页
- 居中卡片式登录框（宽 400px）
- 左侧品牌色装饰条
- 公司 Logo + 系统名称
- 账号/密码/登录按钮

### 7.2 管理端首页（Dashboard）
- 顶部统计卡片行（4-5 个指标卡）
- 中部图表区：结余率趋势图（左 60%）+ 项目分布饼图（右 40%）
- 底部：最近导入记录列表

### 7.3 项目端首页
- 顶部统计卡片（当前项目的指标）
- 中部图表区：该项目的结余率趋势图
- 底部：该项目最近导入记录

### 7.4 数据导入页
- 步骤条：选择项目 → 选择数据类型 → 上传文件 → 预览确认 → 导入完成
- 上传区域：虚线边框拖拽区 + 点击选择
- 校验结果：成功/失败行高亮显示

### 7.5 报表分析页
- 筛选条件栏（时间范围、项目、甲方单位）
- 图表区（趋势图 + 对比柱状图）
- 数据明细表（分页、排序）
- 导出按钮（亮橙）
