# 钢筋精细化管理平台 — 操作手册

> 版本：v1.0 | 日期：2026-06-21

---

## 一、系统概述

钢筋精细化管理平台是一个 B/S 架构的 Web 应用，为建筑公司提供：

- **钢筋台账管理**：进场量、调拨量、措施筋、盘点量、废料量、模型量六大台账
- **配料单管理**：Excel 批量导入、按构件/楼层/直径统计、施工状态跟踪
- **数据分析**：结余率计算、项目盈亏分析、跨项目对比
- **可视化看板**：统计卡片、树状图、饼图/柱状图、动态数据表格

### 访问地址

```
http://<服务器IP>:5000
```

### 默认账号

| 角色 | 初始账号 | 初始密码 | 权限 |
|------|---------|---------|------|
| 管理员 | 联系管理员获取 | — | 全部功能 |
| 工程师 | 联系管理员获取 | — | 授权项目 |

> 首次登录后请立即修改密码

## 二、管理员操作指南

### 2.1 管理看板（首页）

登录后进入管理控制台，包含四个区域：

| 区域 | 内容 |
|------|------|
| **统计卡片** | 活跃项目数、甲方单位数、承接公司数、活跃用户、配料单构件数、总吨数 |
| **筛选栏** | 按承接公司或甲方单位筛选数据 |
| **项目结构树** | 所有项目树形展示，含构件数和吨数 |
| **饼图** | 配料单构件类型分布 + 施工状态分布 |
| **数据表格** | 项目数据汇总表，含配料单吨数、进场量、结余率等 |

### 2.2 基础数据管理

#### 甲方单位管理

`侧边栏 → 基础数据 → 甲方单位`

- 点击 **新建甲方单位** → 输入名称 → 保存
- 列表页可编辑或删除已有单位

#### 承接公司管理

`侧边栏 → 基础数据 → 承接公司`

- 点击 **新建承接公司** → 填写名称/联系人/电话 → 保存

#### 项目管理

`侧边栏 → 基础数据 → 项目管理`

- **新建项目**：选择甲方单位 + 承接公司（可选）→ 输入项目名称 → 保存
- **编辑项目**：修改项目信息、更改状态（活跃/归档）
- **删除项目**：仅在没有关联数据时可删除

### 2.3 配料单管理

#### 批量导入

`侧边栏 → 配料单管理 → 配料单 → 选择项目 → 批量导入`

1. 准备好 Excel 文件（.xlsx 或 .xls 格式）
2. 文件名格式：`项目名 楼层 区域 构件类型.xlsx`（如 `跨建-CIC4#科研中心 第1层 C区 柱.xlsx`）
3. 拖拽文件到上传区域或点击选择
4. 点击"开始导入"
5. 查看导入结果

> Excel 格式要求：第一行为表头（构件名称 + 6/8/10/12/.../32mm 直径列），数据行每行为一个构件

#### 数据汇总

`侧边栏 → 配料单管理 → 配料单 → 选择项目 → 数据汇总`

- **左侧树形导航**：楼栋→楼层→区域，点击节点筛选
- **右侧表格**：构件列表，按直径分列显示重量（吨），颜色区分施工状态
- **施工状态**：每行下拉改状态（未施工→正在施工→已浇筑），支持批量修改
- **图表**：顶部统计卡片 + 进度条 + 饼图（按类型）+ 柱状图（按楼层）
- **导出**：点击"导出Excel"下载汇总表（含构件汇总和楼层汇总两个Sheet）

#### 构件明细

点击表格中的 👁️ 按钮进入构件详情：

- 左侧：构件信息（名称/类型/状态/总重量）
- 右侧：钢筋明细表（直径/重量/根数/单长/总长/来源文件）

#### 批次审核

`侧边栏 → 配料单管理 → 批次审核`

- 查看所有导入批次，按状态筛选
- 点击"审核"弹窗 → 填写意见 → 通过/驳回

### 2.4 数据分析

#### 跨项目盈亏

`侧边栏 → 数据分析 → 项目盈亏分析`

- 统计卡片：所有项目总收入/总成本/总净利润
- 对比表格：每个项目的节超收益、成本、净利润、利润率
- 点击 👁️ 进入单个项目的盈亏详情

### 2.5 钢筋台账

#### 料单审核台账

`侧边栏 → 数据审核 → 料单审核台账`

- 查看所有料单，按状态筛选（待审核/已审核/已通过/已驳回）
- 点击"审核"进入详情 → 填写意见 → 通过/驳回

## 三、工程师操作指南

### 3.1 项目列表

登录后首页显示被授权的项目列表：

- 每个卡片显示：项目名称、甲方/承接公司、配料单构件数/吨数
- 点击按钮：**详情** / **效果图** / **料单** / **成本** / **盈亏**

### 3.2 项目概况

点击项目卡片"详情"按钮：

- 项目信息（甲方/承接公司/状态/描述）
- 钢筋台账关键指标（进场量/调拨量/废料量/结余率）
- 配料单统计（楼栋数/构件总数/总吨数/已浇筑吨数）
- 效果图展示（支持上传管理）

### 3.3 效果图管理

`项目详情 → 管理效果图`

- 上传：选择图片文件（JPG/PNG/GIF/WEBP）→ 上传
- 设封面：点击"设封面"按钮
- 删除：点击 🗑️ 按钮

### 3.4 料单台账

`项目详情 → 料单`

- **新建料单**：填写日期/批次/使用部位/劳务队 → 添加明细（规格/直径/下料长度/根数/单重）
- **提交审核**：点击"提交审核"后将料单发送至管理员
- **查看详情**：查看料单信息和明细表

### 3.5 项目成本

`项目详情 → 成本`

- 顶部按费用类别（人工/材料/机械/运输/管理/其他）展示汇总卡片
- **录入成本**：填写日期/类别/项目/金额/凭证号
- 可编辑或删除已有记录

### 3.6 项目盈亏

`项目详情 → 盈亏`

- 参数设置：钢筋节约定价（元/吨）、其他收入
- 数据明细：节约量 → 节超收益 → 总收入 → 总成本 → 净利润 → 利润率
- 自动根据成本数据和结余率计算结果更新

### 3.7 配料单操作

同管理员配料单操作，但仅限授权的项目。

## 四、常见问题

### Q1: 导入 Excel 后显示"成功 0 条"

**原因**：Excel 格式不符合要求。

**解决**：
1. 检查表头是否包含直径列（6, 8, 10, 12, 14, 16, 18, 20, 22, 25, 28, 32）
2. 检查第一列是否为构件名称
3. 确保直径列的重量数据为数字格式

### Q2: 页面显示乱码

**解决**：刷新浏览器（Ctrl+F5），或更换为 Chrome/Edge 浏览器。

### Q3: 无法访问

**解决**：
1. 确认服务器 Docker 容器正在运行：`docker ps`
2. 确认阿里云安全组已放行 5000 端口
3. 确认服务器防火墙未拦截

### Q4: 忘记密码

**解决**：联系管理员重置密码。
