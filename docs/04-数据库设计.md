# 数据库设计

> 版本：v2.0 | 日期：2026-06-21 | 状态：基于真实模板重设计

---

## 1. 概述

基于8个真实 Excel 模板解析结果，数据库设计覆盖六大核心数据表 + 支撑表。结余率公式涉及10个指标，数据来自6张子表。

### 结余率核心公式（来自 `10、结余率分析` 总表）

```
施工图预算量 ⑤ = 主体结构量① + 措施筋② + 非实体量③ - 二构量④
使用量     ⑪ = 进场量⑥ - 剩余量⑦ - 废料量⑧ - 已施工二构量⑨ - 调拨量⑩
节约量     ⑫ = ⑤ - ⑪
结余率     ⑬ = ⑫ ÷ ⑤ × 100%
```

## 2. ER 图

```
users ──< user_projects >── projects ──< client_units
                                     │
       ┌─────────┬─────────┬─────────┼─────────┬─────────┬──────────┐
       │         │         │         │         │         │          │
  incoming   transfer  measure_  inventory   waste    model_    secondary_
             (调拨量)   rebar    (盘点量)  (废料量)  quantity   structure
            (进场量)  (措施筋)                      (模型量)   (二构量)
                                  │
                            project_analysis (汇总分析，存储 ①~⑬ 计算结果)
```

## 5. 视图：项目汇总

```sql
CREATE VIEW v_project_summary AS
SELECT 
    p.id AS project_id,
    p.name AS project_name,
    cu.name AS client_unit,
    COALESCE(pa.balance_rate, NULL) AS balance_rate,
    COALESCE(pa.budget_qty, 0) AS budget_qty,
    COALESCE(pa.usage_qty, 0) AS usage_qty,
    COALESCE(pa.saved_qty, 0) AS saved_qty,
    COALESCE(pa.incoming_qty, 0) AS incoming_qty,
    COALESCE(pa.transfer_qty, 0) AS transfer_qty
FROM projects p
LEFT JOIN client_units cu ON p.client_unit_id = cu.id
LEFT JOIN project_analysis pa ON p.id = pa.project_id;
```
